开发工具链
  1. Visual Studio

    世界上最好用的开发工具，可以用它里面的F#语言开发数据分析解决方案，以前的2017年老版本也有R语言开发工具模块。还有最流行的数据分析语言Python。使用免费的社区版就可以开发。

  2. Visual Studio for Mac

    在Mac电脑上开发数据分析解决方案，不过功能相比Windows版本少。

  3. Visual Studio Code

    一个跨平台的开发工具，可以在Windows,Mac,Linux三大操作上面运行，具有及其丰富的扩展，使用它上面的热门扩展。如C,C#,C++,Java,Python,R语言等扩展模块，可以开发复杂的数据分析解决方案。

  4. Git

    非常流行的源代码管理工具，具有跨平台能力。结合CLI命令行使用，可以具备强大的源代码管理能力，同时它的常用功能也有GUI界面，要想发挥它的全部功能，还得使用CLI命令行操作。该软件原来是管理Linux操作系统开源代码的，因此它具备大型复杂软件项目代码管理能力。
